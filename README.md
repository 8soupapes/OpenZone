# OpenZone

OpenZone is an open-source home HVAC zoning system designed for ducted heat pumps.

The project is built around an ESP32 and provides intelligent room-by-room climate control using motorized dampers, temperature sensors, and a web-based interface.

## Main Features

- Room-by-room temperature control
- Automatic motorized damper management
- Ethernet-based local web interface
- Midea heat pump integration (bus reverse engineering)
- EDF Tempo energy optimization
- Fireplace heat recovery mode
- Airflow and static pressure management
- Filter monitoring
- Data logging and historical trends
- Automatic fail-safe mode
- Modular architecture for future extensions

## Hardware

- ESP32 / ESP32-S3
- KinCony KC868 controller
- DS18B20 temperature sensors
- 24V motorized dampers
- Ethernet (RJ45)
- Optional microSD storage
- Logic analyzer support for protocol development

## Project Goals

OpenZone aims to provide a professional, fully local, and open-source alternative to proprietary HVAC zoning systems.

The project focuses on:
- Comfort
- Energy efficiency
- Reliability
- Expandability
- Complete local control without cloud dependency

This project is currently under active development.
